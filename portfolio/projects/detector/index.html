Présentation

Ce projet consiste à créer un détecteur d’obstacles utilisant un capteur ultrasonique HC-SR04, une LED et un buzzer.

  • LED éteinte si aucun obstacle proche
  • Bip lent si un obstacle approche
  • Bip rapide + LED si obstacle très proche

C’est un prototype fondamental pour comprendre l’acquisition de données et la logique embarquée.

Objectifs du projet

  • Utiliser le capteur HC-SR04 (temps → distance)
  • Comprendre pulseIn()
  • Programmer LED + buzzer selon la distance
  • Construire un prototype fonctionnel

Matériel utilisé

  • Arduino Uno
  • Capteur HC-SR04
  • LED + résistance 220 Ω
  • Buzzer actif
  • Breadboard + jumpers
  • USB + Arduino IDE

Schéma & architecture

Le capteur HC-SR04 utilise deux broches : Trig pour l’émission et Echo pour la réception.

Code Arduino complet


const int trigPin = 10;
const int echoPin = 11;
const int buzzer = 12;
const int led = 13;

long duree;
int distance;

void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  pinMode(buzzer, OUTPUT);
  pinMode(led, OUTPUT);
  Serial.begin(9600);
}

void loop() {

  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);

  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  duree = pulseIn(echoPin, HIGH);
  distance = (duree * 0.034) / 2;

  Serial.print("Distance : ");
  Serial.print(distance);
  Serial.println(" cm");

  if (distance > 40 || distance <= 0) {
    noTone(buzzer);
    digitalWrite(led, LOW);

  } else if (distance > 20 && distance <= 40) {
    tone(buzzer, 1500);
    digitalWrite(led, LOW);
    delay(200);

  } else {
    tone(buzzer, 2000);
    digitalWrite(led, HIGH);
    delay(100);
  }
}

Résultats

  • Détection fiable entre 5 et 80 cm
  • Buzzer réactif selon la distance
  • Montage stable
